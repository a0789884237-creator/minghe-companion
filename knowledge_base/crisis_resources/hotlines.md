# 心理危机干预资源指南

## 什么是心理危机？

心理危机是指个体在面临重大生活事件或持续压力时，超出其应对能力，出现严重情绪困扰、功能受损的状态。常见表现包括：
- 强烈的绝望感
- 无法控制的情绪
- 失眠、不吃不喝
- 有自我伤害的想法
- 行为明显异常

## 危机信号识别

### 言语信号
- "活着没意思"
- "想死"
- "大家没有我会更好"
- "我想结束一切"
- 谈论具体的自杀方法
- 与亲友告别

### 行为信号
- 突然的平静或情绪好转（可能是决定后释然）
- 赠送重要物品
- 整理个人物品
- 写下遗书
- 远离亲友

### 情绪信号
- 持续的极度绝望
- 深深的愧疚或羞耻
- 无法摆脱的愤怒
- 极度的孤独

## 应对方法

### 如果你有自杀想法

1. **寻求帮助**
   - 告诉信任的人你的感受
   - 拨打心理援助热线

2. **远离危险物品**
   - 把刀、药物等交给别人保管
   - 删除网购记录

3. **制定安全计划**
   - 列出可以帮助你的人
   - 写下的就医步骤

4. **分散注意力**
   - 做运动
   - 听音乐
   - 打电话给朋友

5. **紧急情况**
   - 拨打120或110
   - 前往最近医院急诊

### 如果身边的人有危机信号

1. **认真对待**
   - 不要轻视或忽视
   - 相信你的直觉

2. **倾听和沟通**
   - 找一个安静的时间
   - 表达你的关心
   - 认真倾听，不评判
   - 问清楚他们的想法

3. **不要**
   - 不要说"不要想不开"
   - 不要承诺保密（需要告知专业人）
   - 不要让他们独处

4. **寻求帮助**
   - 联系心理专业人士
   - 联系家人
   - 拨打危机热线

## 自杀预防原则

### TALK原则
- **T**ell: 告诉对方你在乎他
- **A**sk: 直接询问自杀想法
- **L**isten: 倾听而不评判
- **K**eep Safe: 确保安全，寻求帮助

## 记住

- 自杀是可以预防的
- 寻求帮助是勇敢的表现
- 绝大多数危机都是暂时的
- 你的生命很重要
- 有人愿意帮助你
- 痛苦不会永远持续
