# 认知行为疗法（CBT）入门

## 什么是CBT？

认知行为疗法（Cognitive Behavioral Therapy，CBT）是一种结构化、目标导向的心理治疗方法。它基于"认知-情绪-行为"三者的相互关联，通过改变负性思维模式和不适应行为，来改善情绪状态。

## CBT的核心原理

### 1. 认知三角
```
      想法（认知）
         ↕
情绪 ←————————→ 行为
```

- **认知**：我们对事件的解释和评价
- **情绪**：由认知引发的情感反应
- **行为**：情绪驱动下的行动

### 2. 核心观点
- 事件本身不导致情绪，而是我们对事件的解读导致情绪
- 改变思维模式可以改变情绪反应
- 改变行为可以打破恶性循环

## CBT的基本技术

### 1. 思维记录表

当有负面情绪时，记录以下内容：

| 项目 | 内容 |
|------|------|
| 情境 | 发生了什么？ |
| 自动思维 | 我当时在想什么？ |
| 情绪 | 我感受到什么？（1-10分） |
| 身体感觉 | 身体有什么反应？ |
| 行为 | 我做了什么？ |

### 2. 识别认知扭曲

常见的思维误区：

- **全或无思维**：用"总是"、"从不"等绝对化词汇
- **灾难化**：把问题想得极其严重
- **读心术**：认为别人一定怎么想
- **情绪推理**：感觉是这样就是事实
- **应该陈述**："我应该"、"我必须"
- **个人化**：把不是自己的责任揽上身

### 3. 苏格拉底式提问

通过提问帮助自己发现问题：

- "支持这个想法的证据是什么？"
- "反对这个想法的证据是什么？"
- "有没有其他解释？"
- "一年后这还重要吗？"
- "我会对朋友这样说吗？"

## 常用CBT练习

### 1. 认知重构

步骤：
1. 识别自动思维
2. 找出认知扭曲
3. 收集证据
4. 形成更平衡的想法

示例：
- 原思维："我演讲搞砸了，大家一定觉得我很差劲"
- 证据支持：确实有停顿
- 证据反对：大多数人很支持，有人还鼓掌
- 新思维："虽然有小失误，但整体表现还不错，大家很支持"

### 2. 行为激活

帮助打破"抑郁-回避-更多抑郁"的循环：

1. 列出愉快活动
2. 列出有意义活动
3. 从小开始，每天做一件
4. 记录活动后的情绪

### 3. 暴露疗法

面对恐惧，逐步脱敏：

1. 列出恐惧分级清单
2. 从最低级开始
3. 逐步面对，每次坚持直到焦虑下降
4. 重复练习

## CBT的适用人群

CBT对以下问题特别有效：
- 抑郁症
- 焦虑障碍
- 恐惧症
- 强迫症
- 失眠
- 压力管理
- 低自尊

## 注意事项

- CBT需要积极参与和练习
- 效果需要时间显现
- 如有严重心理问题，建议寻求专业心理咨询师帮助
- CBT可以与其他治疗方法结合使用
